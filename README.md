# 🦜 Guia Vip - Guia Comercial

Guia comercial de cidades para encontrar empresas locais.

## Arquivos incluídos

- `index.html` — Página principal
- `styles.css` — Estilos visuais
- `app.js` — Lógica da aplicação
- `manifest.json` — Configuração PWA
- `sw.js` — Service Worker
- `city-card-bg.jpg` — Imagem dos cards de cidades
- `special-card-bg.jpg` — Imagem dos cards especiais

## Hospedagem

Faça upload de todos os arquivos para qualquer servidor web.

## Tecnologia

- Progressive Web App (PWA)
- Firebase Firestore
- Compatível com Android e iOS
